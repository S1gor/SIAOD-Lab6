## Сравнение методов сортировки пузырьком (bubble) и сортировки выбором (selection).
### Количество перестановок, чтобы отсортировать целиком:
состояние отсортированности массива | сортировка пузырьком | сортировка выбором
:----------------------------------:|:--------------------:|:-------------------:
неотсортирован только один элемент | равное месту, на котором находится данный элемент(считаем от нуля) | равное месту, на котором находится данный элемент(считаем от нуля)
